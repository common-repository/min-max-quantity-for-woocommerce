== Description ==
CI WooCommerce Minimum Maximum Quantity allows you to define the minimum and maximum allowable product quantities per product or all products of your shop.

== Installation ==

1. Unzip the downloaded zip file.
2. Upload the plugin folder into the `wp-content/plugins/` directory of your WordPress site.
3. Activate `CI WooCommerce Minimum Maximum Quantity  ` from Plugins page
